## Map Data
### size
size = list[x, y]
This is list that show map size.
The size is the number of block

### row and column
row, column = list[1|2]
1 mean wide block
2 mean narrow block

### map
map is  2-Dimensional List composed by 0 and 1, 2. 3
0 mean space
1 mean thick wall
2 mean thin wall
3 mean door
